# Cavinior — Pure Indulgence

A cinematic, 3D scroll-driven web experience for **Cavinior**, a luxury chocolate brand established in 1924. This project showcases the intersection of high-end artisanal craftsmanship and modern web technology.

![Cavinior Preview](https://github.com/user-attachments/assets/0aa67016-6eaf-458a-adb2-6e31a0763ed6)

## ✨ Key Features

- **Cinematic 3D Scroll Animations:** A high-performance canvas-based sequence that reacts to the user's scroll, revealing the heritage and craft of Cavinior chocolate.
- **Luxury Loading Screen:** A custom-engineered entrance experience featuring animated gold ornamental lines, a delicate progress bar, and elegant brand reveals.
- **Dynamic Theme Engine:** Seamless switching between a deep "Chocolate Dark" mode and a sophisticated "Cream Linen" light mode.
- **GSAP Interactivity:** Butter-smooth micro-interactions, staggered reveals, and scroll-triggered animations powered by Greensock.
- **Editorial Typography:** A refined pairing of *Playfair Display* for legacy and *Outfit* for modern precision.

## 🛠️ Tech Stack

- **Frontend:** [React 19](https://react.dev/)
- **Animation:** [GSAP 3](https://greensock.com/gsap/) (ScrollTrigger)
- **Styling:** [Tailwind CSS 4](https://tailwindcss.com/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Build Tool:** [Vite](https://vitejs.dev/)

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (Latest LTS recommended)

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Sandip-Basak/3D-Animation-Website.git
    cd 3D-Animation-Website
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

4.  **Open the app:**
    Navigate to `http://localhost:3000` (or the port specified in your terminal).
